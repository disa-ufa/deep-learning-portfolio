# Нейро-помощник вайб-программиста

Учебно-практический RAG-проект: нейро-сотрудник для фриланс-разработчика, который анализирует заявки клиентов и помогает подготовить профессиональный ответ.

## Задача

Создать нейро-сотрудника, который по тексту клиентской заявки:

- определяет тип проекта;
- выделяет, что хочет клиент;
- формирует уточняющие вопросы;
- предлагает техническое решение;
- оценивает сложность, сроки и стоимость;
- предупреждает о рисках;
- готовит готовый ответ заказчику.

## Что реализовано

- загрузка базы знаний из Google Docs;
- структурирование базы знаний по двухуровневой схеме;
- разбиение знаний на фрагменты для RAG;
- создание эмбеддингов через OpenAI Embeddings;
- хранение и поиск фрагментов в ChromaDB;
- генерация ответа через OpenAI Chat Completions;
- Gradio-интерфейс с кнопками обучения и запроса;
- контрольная проверка работы без интерфейса.

## Архитектура

```text
Google Docs knowledge base
        ↓
Text download
        ↓
Structured fragments: H1 / H2 / Fragment
        ↓
OpenAI Embeddings
        ↓
ChromaDB vector store
        ↓
Similarity search
        ↓
Prompt + retrieved context
        ↓
LLM answer
        ↓
Gradio UI
```

## Результат

Нейро-сотрудник успешно загружает базу знаний, находит релевантные фрагменты и формирует структурированный ответ по заявке клиента.

Пример тестовой заявки: интеграция Instantly.ai с Google Таблицей.

Модель сформировала:

- тип проекта;
- список уточняющих вопросов;
- возможное техническое решение;
- оценку сложности;
- примерные сроки и стоимость;
- риски;
- готовый текст ответа клиенту.
