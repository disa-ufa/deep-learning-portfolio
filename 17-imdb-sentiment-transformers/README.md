# IMDB Sentiment Classification

## Описание
Проект по бинарной классификации отзывов IMDB на положительные и отрицательные.

## Цель
Построить модель, обеспечивающую:
- accuracy на validation выше 90%;
- accuracy на test выше 88%;
- по возможности выйти на повышенный уровень качества.

## Подход
Использован transfer learning с предобученной трансформерной моделью для классификации текста.
Подбор конфигурации выполнялся по validation, а test использовался только для финальной оценки.

## Результаты
- Validation Accuracy: **0.9324**
- Test Accuracy: **0.9354**

## Используемые технологии
- Python
- TensorFlow
- Hugging Face Transformers
- Google Colab
