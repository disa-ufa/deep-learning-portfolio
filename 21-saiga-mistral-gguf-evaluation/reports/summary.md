# Краткий отчёт

## Цель

Получить практические навыки работы с большими языковыми моделями, Hugging Face, квантованными GGUF-моделями и базовой оценкой качества ответов LLM.

## Используемая модель

В работе использовалась русскоязычная модель IlyaGusev/saiga_mistral_7b_gguf.

Файл модели: model-q4_K.gguf.

Этот файл выбран как компромисс между качеством генерации и требованиями к памяти.

## Что было сделано

1. Создана учётная запись Hugging Face.
2. Получен Hugging Face token.
3. Проверен доступ к репозиторию модели Saiga GGUF.
4. Проверен доступ к датасету ru_turbo_saiga.
5. Загружена квантованная модель model-q4_K.gguf.
6. Модель запущена в Google Colab на GPU Tesla T4.
7. Сгенерированы 3 шутки на русском языке.
8. Загружен датасет ru_turbo_saiga.
9. Подготовлены 5 тестовых примеров.
10. Получены ответы модели Saiga.
11. Проведено сравнение ответов модели с эталонными ответами через cosine similarity.

## Результат

Accuracy: 80%

Порог похожести: similarity > 0.5.

Из 5 примеров 4 ответа модели прошли выбранный порог.

## Ограничения

Оценка является учебной:

- использовано только 5 примеров;
- датасет мог использоваться при обучении Saiga;
- cosine similarity не проверяет фактическую корректность ответа;
- для полноценной оценки нужно больше примеров и ручной анализ ошибок.

## Вывод

Работа успешно демонстрирует полный цикл использования локальной LLM:

- загрузка модели;
- инференс;
- генерация текста;
- работа с датасетом;
- автоматизированная оценка качества.
