# Saiga Mistral 7B GGUF: генерация и оценка качества

Учебно-практический проект по работе с локальной русскоязычной LLM: загрузка квантованной модели Saiga Mistral 7B в формате GGUF, генерация текста и базовая оценка качества на датасете ru_turbo_saiga.

## Задача

Получить практические навыки работы с большими языковыми моделями и Hugging Face:

- создать и использовать Hugging Face token;
- загрузить русскоязычную модель Saiga;
- сгенерировать 3 шутки;
- использовать квантованную GGUF-версию модели;
- провести простую оценку качества на датасете ru_turbo_saiga.

## Что реализовано

- авторизация в Hugging Face из Google Colab;
- проверка доступности репозиториев модели и датасета;
- загрузка IlyaGusev/saiga_mistral_7b_gguf;
- использование файла model-q4_K.gguf;
- запуск модели через llama-cpp-python;
- генерация 3 шуток на русском языке;
- загрузка датасета IlyaGusev/ru_turbo_saiga;
- подготовка 5 пар question -> reference_answer;
- генерация ответов Saiga на тестовые вопросы;
- оценка качества через cosine similarity;
- расчёт accuracy по порогу similarity > 0.5.

## Результаты

Модель была успешно запущена в Google Colab на GPU Tesla T4.

Использовалась квантованная версия:

model-q4_K.gguf

Модель сгенерировала 3 шутки на русском языке:

- про программиста;
- про нейросети и фриланс;
- про Hugging Face и Google Colab.

Для оценки было подготовлено 5 примеров из датасета ru_turbo_saiga.

Итоговая метрика:

Accuracy: 80%

Порог similarity: 0.5

Количество примеров: 5

## Ограничения оценки

Оценка является учебной и демонстрирует принцип проверки качества LLM, но не является полноценным benchmark:

- модель могла видеть ru_turbo_saiga при обучении;
- использовано только 5 примеров;
- cosine similarity оценивает смысловую близость, но не гарантирует фактическую правильность;
- часть ответов может быть похожей по смыслу, но содержать фактические ошибки.
