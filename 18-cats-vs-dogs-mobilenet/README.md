# Cats vs Dogs Classification with MobileNet

## Описание
Проект по классификации изображений кошек и собак с использованием полного датасета, data augmentation и transfer learning.

## Цель
Построить модель на основе MobileNet и добиться высокой точности на test.

## Подход
- использование полного датасета;
- аугментация изображений;
- MobileNet как предобученная база;
- fine-tuning верхних слоёв модели.

## Результаты
- Validation Accuracy: **0.9833**
- Test Accuracy: **0.9792**

## Используемые технологии
- Python
- TensorFlow / Keras
- Google Colab
